# Shreeji Enterprise Services

Official website for Shreeji Enterprise Services - A leading manpower supply and recruitment company.

## About

Shreeji Enterprise Services is a premier manpower supply company based in Rajkot, Gujarat, India. We specialize in three main service areas:

1. **Delivery & Logistics Personnel** - Recruitment for delivery platforms like Swiggy, Zomato, Zepto, Blinkit, Rapido, Porter, and Expressbees
2. **BPO & BFSI Staffing** - Mid-level positions for voice/non-voice, process support, customer care, and operations
3. **IT Recruitment** - Global recruitment services for IT companies

## How to Run

Simply open `index.html` in a web browser, or serve it using any web server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx http-server
```

Then visit `http://localhost:8080` in your browser.
